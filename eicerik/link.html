Bağlantı Biçimlendirme Link Biçimlendirme

Bağlantı Bİçimlendirme

HTML bağlantılarını (Link) CSS kullanarak biçimlendirebiliriz. Biçimlendirirken önceden anlattığımız color, background, font vb. özellikleri kullanabiliriz.

Linkler yazı, resim veya buton şeklinde biçimlendirilebilirler.Aşağı farklı şekillerde biçimlendirilmiş linkleri inceleyebilirsiniz.


Okulumuz Okulumuz Okulumuz

Örnek Kod

        
	a:link{						
    color:blue;					
	}
	
	a:visited{					
    color:green;				
	}
	
	a:hover{					
    text-decoration:underline;	
	}		
	a:active{					
    color:black;				
	}
