display Özelliği

CSS display özelliği HTML elementlerinin bir web sayfasında nasıl yerleştirileceğini kontrol etmemize yarayan çok önemli bir özelliktir. Her HTML elementinin kendine özel varsayılan display özelliği vardır. Display değeri blok (block) ve satır içi(inline) olmak üzere iki çeşittir.

Display özelliği blok olan elementler yeni bir satırda başlar ve genişliği satırı kaplar.Bu bir div elementi, div blok seviyeli bir elementtir.

Display özelliği satır içi olan elementler yeni satırda başlamaz genişliği içeriğinin genişliği kadar kalır. Bu satırda img etiketleri kullandık. img satır içi elementtir.

display:none özelliği ise elementin görüntülenmemesini sağlar. Aşağıda display:none; değeri verilmiş bir div elementimiz var. Ama göremiyoruz :)

Display özelliği blok olan elementler yeni bir satırda başlar ve genişliği satırı kaplar.Bu bir div elementi, div blok seviyeli bir elementtir.

Aşağıda varsayılan display özelliği blok (block) olan bazı HTML etiketlerini görebilirsiniz.

  • div
  • form
  • h1-h6
  • header
  • hr
  • li
  • main
  • ol
  • p
  • table
  • pre
  • ul

Aşağıda varsayılan display özelliği satır içi (inline) olan bazı HTML etiketlerini görebilirsiniz.

  • a
  • b
  • br
  • button
  • i
  • img
  • input
  • label
  • select
  • span
  • strong
  • textarea

Örnek Kod

        
		
	img{
    display:block;	
	}
	
	
	li{
    display:inline;	
	}
	
	
	div{
	display:none;
	}
