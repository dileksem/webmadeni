Liste Bİçimlendirme

  • HTML listelerini
  • CSS kullanarak
  • biçimlendirebiliriz.
  • CSS ile
  • listelerin
  • işaretlerini değiştirebiliriz
  1. İstediğimiz
  2. bir resmi
  3. madde işareti
  1. olarak
  2. belirleyebiliriz.
  3. Listeleri menü haline getirebiliriz.
  • list-style-position:outside
  • özelliği ile madde işaretlerini
  • liste elementinin dışına konumlandırabiliriz.
  • list-style-position:inside
  • özelliği ile madde işaretlerini
  • liste elementinin içine konumlandırabiliriz.
  • Listeleri renkler kullanarak biçimlendirebiliriz.
  • Burada arkaplan rengi ul için mavi,li için sarı
  • padding ul için 30px, li için 5px verildi.
  • Burada arkaplan rengi ul için pembe ,li için sarı
  • padding ul için 25px verildi.
  • li margin değerine 10px vererek birbirinden uzaklaştırmış olduk.

Örnek Kod

        
		
	ol{	
    list-style-type: upper-roman;	
	list-style-position:outside;	
	background:#ffb3ff; 
	padding: 25px;	
	}	
	
	li{		
	background:#ffffcc; 			
	margin:10px;	
	}
